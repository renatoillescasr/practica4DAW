GITHUB

1. colocarse enla carpeta que contiene los archivos con
   $ cd [ruta]

2 inicializar un respositorio local
   $ git init

3. ver Estado del repositorio
   $ git status [rojo archivos no subidos o modificados]

4. para agregar archivo
   $ git add [-A][.](agrega todos) o [archivo]
   
   - si corremos un $ git status, saldra de color verde con el cambio      modificado

5 copiar el url del repositorio web del githu y agregarlo
   $ git remote add origin http..

6 agregamos texto al cambio hecho
   $ git commit -m "mensaje"

7 sincronizar nuestro escritorio remoto con el local
   $ git push origin master
 
8. crear rama
$ git branch gh-pages

9 subir a la rama
$ git branch

10 actualizar la rana
$ git push origin master gh-page
